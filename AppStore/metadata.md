# JotJot - App Store 信息

## 应用名称
JotJot

## 副标题
想到就记，记完就走

## 关键词
笔记,灵感,快速记录,闪念,备忘录,iCloud,同步,Markdown

## 描述
JotJot 是一款极简的灵感捕捉工具。

**核心理念：** 想到就记，记完就走。

### 主要功能

🚀 **极速启动**
- Mac: ⌘⇧J 全局快捷键，随时呼出
- iOS: 打开即写，零等待

✨ **优雅体验**
- 滑动切换笔记，流畅自然
- Markdown 实时预览
- 链接自动预览

🎤 **语音输入**
- 说话即记录
- 自动转文字

🤖 **智能合并**
- AI 识别相似笔记
- 一键合并整理

☁️ **无缝同步**
- iCloud 自动同步
- Mac + iOS 双端

📱 **桌面小组件**
- 快速记录入口
- 一触即达
