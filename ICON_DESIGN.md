# JotJot App 图标设计说明

## 设计理念

**核心概念：闪念的瞬间**

JotJot 是一款「闪念捕捉」应用，图标设计围绕「灵感闪现的那一刻」展开。

## 视觉元素

### ⚡ 闪电符号
- 代表「闪念」—— 灵感如闪电般转瞬即逝
- 极简几何造型，一眼可识别
- 向下的动势暗示「捕捉、落地」

### 🎨 渐变配色
```
#6366F1 (靛蓝) → #8B5CF6 (紫罗兰) → #F59E0B (琥珀橙)
```

| 颜色 | 含义 |
|------|------|
| 靛蓝 | 智者的清晰与专注 |
| 紫罗兰 | 魔法师的变革与奇迹 |
| 琥珀橙 | 照顾者的温暖与能量 |

### 闪电填充
白色到暖黄的渐变，象征灵感的光芒。

## 品牌人格映射

- **魔法师** → 紫色渐变 + 闪电的「变革」意象
- **智者** → 极简设计 + 清晰的几何形态
- **照顾者** → 温暖的橙色收尾 + 柔和阴影

## 技术规格

- **尺寸**: 1024 × 1024 px
- **圆角**: 224 px (iOS 风格)
- **格式**: PNG (透明背景裁切后)
- **源文件**: `Assets/icon.svg`, `generate-icon.js`

## 文件清单

```
Assets/
├── AppIcon.png      # 主图标 1024x1024
└── icon.svg         # 矢量源文件
```
